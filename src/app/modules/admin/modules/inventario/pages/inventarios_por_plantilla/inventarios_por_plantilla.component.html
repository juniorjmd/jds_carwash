Cargue de Inventario por Plantilla

Para realizar el cargue masivo de inventario, siga estos pasos:

  1. Descargue una de las plantillas disponibles a continuación. Puede elegir entre los formatos Excel y CSV:
    • Descargar plantilla Excel
    • Descargar plantilla CSV
  2. Diligencie la plantilla con los datos correspondientes. Asegúrese de no eliminar ni modificar las columnas existentes.
  3. Si utiliza la plantilla Excel, antes de subir el archivo, conviértalo a formato CSV:
    1. Abra el archivo Excel en su editor (por ejemplo, Microsoft Excel).
    2. Haga clic en Archivo > Guardar como.
    3. Seleccione la ubicación donde desea guardar el archivo.
    4. En el campo Tipo, elija CSV (delimitado por comas) (*.csv).
    5. Guarde el archivo.
  4. Seleccione el archivo CSV utilizando el botón a continuación.
  5. Valide el archivo para asegurarse de que no hay errores en los datos. Si se encuentran errores, corrija el archivo y vuelva a cargarlo.
  6. Una vez validado correctamente, podrá procesar el archivo para cargar los datos al sistema.
